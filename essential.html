Besides elements we also use compounds in our everyday life.

A few examples of compounds that we use in our daily lives are:

  • Water ( H2O )
  • Table Salt ( NaCl )
  • Sugar ( C12H22O11 )
  • Soap ( RCOO )
  • Toothpaste ( CaCO3 or NaF )
  • Mouthwash ( H2O2 )
  • Vinegar ( H3COOH )
  • Aspirin ( C9H8O4 )
